# img-to-mnemotablo
Берёт картинку и выплёвывает код на lua для майнкрафта

## Как юзать
-  главный исполняемый файл `main.exe`
-  Аргументы
  - `--img` путь к картинке
  - `--side` Сайд для враппа панели
  - `--fname` Имя панели
  - `--file` Имя файла
  - `--size` Размер панели (0, 1, 2, 3, 4, 5) (2, 4, 8, 16, 32, 64, 128)
